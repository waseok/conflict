  • 새로운 업무 배정 원칙 필요한 경우

4. 갈등 유형별 해결 가이드

가. 업무 소유권 갈등

판단 순서

  1. 법령·지침 확인: 반드시 특정 부서가 해야 하는지
  2. 전문성 검토: 어느 부서가 더 전문적으로 처리 가능한지
  3. 연관성 분석: 기존 업무와의 연계성이 높은 부서
  4. 효율성 고려: 가장 효과적으로 처리할 수 있는 부서

해결 방식: 명확한 소유권 배정 + 업무분장표 즉시 수정

나. 업무 회피 갈등

확인 사항

  1. 실제 업무량: 문서 처리 건수, 회의 참석 횟수 등 객관적 비교
  2. 전문성 요구: 특별한 기술이나 경험이 필요한지
  3. 시급성: 정말 미룰 수 없는 업무인지
  4. 대안 가능성: 다른 방법이나 시기 조정 가능한지

해결 방식: 업무량 재조정 또는 일시적 지원 방안 마련

다. 업무 중복 갈등

분석 포인트

  1. 중복의 원인: 시스템 문제인지, 소통 부족인지
  2. 통합 가능성: 하나의 부서에서 일원화할 수 있는지
  3. 전문성 손실: 통합 시 전문성이 떨어질 위험은 없는지

해결 방식: 주관 부서 지정 + 협조 부서 역할 명확화

라. 업무 과부하 갈등

형평성 검토

  1. 정량적 분석: 처리 공문 수, 담당 업무 건수 비교
  2. 난이도 고려: 단순 반복 vs 고도의 판단 필요 업무
  3. 시기적 집중: 특정 시기에만 몰리는지 상시인지

해결 방식: 업무 재분배 또는 순환 담당제 도입

마. 절차 갈등

효율성 원칙

  1. 최소 단계: 불필요한 경유 부서 최소화
  2. 병렬 처리: 동시 진행 가능한 업무는 병렬로
  3. 명확성: 각 단계별 역할과 기한 명시

5. 회의 운영 (1.1.1.1 원칙)

6. 예방 및 사후관리

갈등 예방책

  • 명확한 업무분장: 경계 업무는 사전에 담당 부서 명시
  • 정기적 업무량 점검: 월 1회 부서별 업무량 모니터링
  • 신규 업무 배정 원칙: 새 업무 발생 시 즉시 소속 부서 지정
  • 소통 채널 상시 개방: 온라인 연구실 통한 수시 의견 교환

7. 핵심 원칙 및 체크리스트

갈등 해결 6원칙

  1. 교육활동 우선: 학생 교육에 미치는 영향을 최우선 고려
  2. 신속 처리: 갈등이 길어질수록 업무 효율성 저하
  3. 공정성: 특정 부서나 개인에게만 부담 집중 금지
  4. 투명성: 결정 과정과 근거를 명확히 공개
  5. 상호 배려: 교육공동체 구성원으로서 서로를 존중하고 협력
  6. 학습: 동일 갈등 재발 방지를 위한 시스템 개선
