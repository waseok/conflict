와석초등학교 업무갈등 조정 실무 매뉴얼 통합판

기존 실무 매뉴얼 + 확장 사례/도구를 조화롭게 반영한 버전

기존 실무 핵심갈등유형해결 프로세스 확장 사례 비교사례 카드체크형 점검표

1. 목적 및 적용범위 (기존 내용 보존)

목적: 교육활동 중심 학교 운영을 위해 부서 간 업무 갈등을 신속하게 해결해 업무 효율성을 높인다.

적용: 전 교직원, 부서 간 업무 갈등 발생 시 즉시 적용한다.

통합 원칙: 기존 운영 매뉴얼의 실무 절차를 유지하면서 확장 사례와 문안 도구를 추가한다.

2. 업무갈등 유형 및 판단기준 (기존 구조 유지)

유형구체적 사례1차 판단기준
업무 소유권"학부모 대상 성폭력예방교육은 보건부인가, 관련 부서인가?"관련 규정, 선례, 전문성
업무 회피"바빠서 못한다" vs "원래 네 업무다" 형태의 책임 전가업무분장표, 시급성, 대체 가능성
업무 중복동일 공문을 여러 부서에서 각각 처리일원화 가능성, 효율성
업무 과부하특정 부서에 업무가 지속적으로 집중업무량 형평성, 처리 역량
절차 갈등협조·결재 순서에 대한 이견규정, 관례, 처리 효율

3. 단계별 해결 프로세스 (기존 내용 + 확장 보강)

갈등 인지 → 24시간 내 1차 협의 → 합의 실패 시 관리자 중재 → 공식 절차

1단계: 당사자 직접 협의

해야 할 것
  • 업무분장표/기존 선례 확인
  • 업무량·일정을 공개적으로 공유
  • 대체안(역할 분담/기한 조정) 제시
하지 말 것
  • 감정적 표현
  • 근거 없는 책임 전가
  • 제3자에게 일방적 전달

2단계: 부서장 또는 교감 중재

  • 쟁점을 1쪽으로 정리해 중재회의 진행
  • 최종결정권자·실무담당자·안내담당자(RACI)를 명확화

3단계: 공식 조정 및 문서화

  • 반복 갈등은 업무분장표 수정으로 구조적 해결
  • 결정사항은 회의록과 공지문으로 남긴다.

4. 확장 사례 비교표 (최신 내용 반영)

사례갈등 유형핵심 처방난이도
늘봄 배정선 혼선운영·배정RACI 재설계, 단일 안내상
학교폭력 자체해결 오판학생보호·안전보호조치 우선, 전담기구 선검토상
민원 단일창구 부재운영·배정교감 중심 단일창구상
정서행동검사 공유범위학생보호·안전최소 접근권한, 비공개 협의체상
근무성적평정 갈등인사·책임근거 설명, 이의신청 절차중상
내부신고 후 보복 의심인사·책임인사 변경 유보, 보호절차 병행상

5. 사례 카드 (검색/필터/문안 복사)

표시 사례: 6건
사례 1. 늘봄 배정 혼선▼

권고배정권자와 안내권자를 분리하지 말고 RACI를 확정한다.

예시 문안“늘봄 배정은 교감이 총괄하고 문의는 전담실이 일괄 접수합니다.”

사례 2. 학교폭력 자체해결 오판▼

권고피해학생 보호를 즉시 시행하고 전담기구 사실확인을 선행한다.

예시 문안“생활지도와 별개로 전담기구 검토 절차를 진행하며 보호조치는 즉시 시행합니다.”

사례 3. 민원 단일창구 부재▼

권고교사 개인 연락 경로를 차단하고 학교 대표 창구로 통합한다.

예시 문안“본 사안은 학교 대표 민원창구로 일원화하며 개인 연락에는 응답하지 않습니다.”

사례 4. 정서행동검사 공유범위 갈등▼

권고열람권한 최소화, 단체 메신저 공유 금지.

예시 문안“검사 결과는 지정 인원만 열람하며 대화방 공유를 금지합니다.”

사례 5. 근무성적평정·성과상여금 갈등▼

권고등급 통보와 근거 설명을 분리하고 이의신청 절차를 명확히 한다.

예시 문안“제출자료를 평정기준과 대조해 재검토하고 결과를 서면 회신하겠습니다.”

사례 6. 내부신고 후 보복 의심▼

권고신고 직후 인사·평가 변경은 관련성 검토 전 유보한다.

예시 문안“신고 이후 인사·평가 변경은 관련성 검토 완료 시까지 유보합니다.”
